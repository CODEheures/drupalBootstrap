# Drupal Bootstrap

> A [Drupal](https://www.drupal.org) configuration with boostrap for fast starting.

## Installation
Clone this repository in your www target folder

```bash
git clone https://github.com/CODEheures/drupalBootstrap.git my_target_folder
```

Install packages
```bash
$ composer install
```
